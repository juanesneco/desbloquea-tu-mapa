Roadmap Visual

Plan Maestro del Sitio

Referencia rápida de todos los ajustes visuales y de interacción que quisiera aplicar al website estático para reflejar la nueva estética Nike/Apple que ya tiene el app de IA.

Branding & Tipografía

  • Tipografía primaria Space Grotesk para titulares (tracking amplio) + Inter para textos largos.
  • Colorimetría Fondo en degradé muy suave (#FFF→#E6E9EF), acentos limas (#C5FF40) y negro puro para jerarquía.
  • Bloques glassmorphism Tarjetas con blur y borde blanco translúcido que reflejen los componentes de la app.

Navegación & Flujo

  • Header modular Logo minimal + CTA “Inicia” alineado a la derecha, sticky con blur y contraste sutil.
  • Secciones Hero (claim + CTA), Filosofía (3 fases), Mapas (5 chips), Herramientas (App/Newsletter/Cartas) y Cierre.
  • Interacción Estados hover con escala ligera y foco en accesibilidad (alto contraste y labels claros).

Módulos Propuestos

Hero

Video/imagen a pantalla completa + tagline “Interpretación simbólica en tiempo real”.

Fases

Cards horizontales con Fase, subtítulo y bullets de transformación.

Mapas

Grid 5 columnas con detalles y CTA “Explorar mapa”.

Galería Viva

Muestras reales de símbolos con tags y fase asignada para inspirar uploads.

Implementación

  • Stack actual Mantener HTML estático pero extraer estilos comunes a un CSS global para compartir tokens.
  • Componentes reutilizables Crear partials para header, footer y hero que puedan inyectarse en cada página del sitio.
  • Paridad app Sincronizar textos de fases/mapas con el nuevo schema (5 mapas, sub-etapas oficiales).
